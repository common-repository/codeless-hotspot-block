== Description ==

Codeless Hotspot Block is a WordPress plugin from Codeless.co, that allows you to easily create hotspots on your website's images. Hotspots are interactive elements that allow you to display additional information when the user clicks on a specific area of an image. With this plugin, you can create engaging and interactive content for your website without writing a single line of code.

== Installation ==

To install Codeless Hotspot Block, follow these steps:

Upload the plugin files to the /wp-content/plugins/codeless-hotspot-block directory using your preferred method (FTP, SFTP, etc.).
Activate the plugin through the 'Plugins' screen in WordPress. You can install block directly from gutenberg editor also.
== Frequently Asked Questions ==

= How do I create a hotspot? =

Add the Codeless hotspot block. Then, select the image and click on the Image position to add a Hotspot. You can after change the content.

= Can I customize the appearance of the hotspots? =

Yes, you can customize the appearance of the hotspots using CSS. The plugin comes with some basic styling options, but you can add your own custom styles to match your website's design.
